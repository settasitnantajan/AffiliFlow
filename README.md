# AffiliFlow

Shopee Video Affiliate Automation - Auto-generate videos, captions, hashtags & product links daily.

## Quick Start

```bash
npm install
cp .env.example .env.local   # แก้ไขค่า API keys
npm run dev                   # http://localhost:8888
```
